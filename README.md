User Registration and Login System
Description

This program is a simple Java-based user registration and login system.
The system allows a user to register by entering their personal details such as first name, last name, username, password, and cellphone number.

After successful registration, the user is required to log in using the same credentials.
The system validates the input using specific rules and ensures that the login details match the registered information.


 Technologies Used

* Java
* JOptionPane (for user input and output)
* JUnit (for testing using assertTrue and assertFalse)


Features

* Username validation (must contain "_" and be ≤ 5 characters)
* Password complexity validation
* Cellphone number validation (+27 format)
* Login authentication using stored details
* Looping until correct input is entered
* Unit testing using JUnit
